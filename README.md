##### PREREQUISITES
* Python 3.x 
* pip3
* git

##### PYTHON 3 PREREQUISITES
* aiohttp
* asyncio

## Installation
```sh
$ git clone https://github.com/LanangAkira/OJS-ShellFinder
$ cd OJS-ShellFinder
$ pip3 install -r requirements.txt
$ python3 ojs.py
```
##### Update Wordlist
```sh
$ apt install wget
$ python update.py
```
#### Note
